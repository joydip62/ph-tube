Discuss the scope of var, let, and const

var : A var variable can have the same meaning in all scopes, and can be automatically declared in the global scope if it is declared in the function scope. It allows hosting, which means we can declare a var at the beginning of any scope.


let : A let variable is limited to block scope, it does not provide hosting scope, and the value must be used in the scope in which we declared it.


const : A const variable is limited to block scope and its value cannot be changed once it is declared.

Tell us the use cases of null and undefined

null : null basically represents an empty value which is generally used when a variable or object has no value.


undefined : If a variable is not declared, the value of that variable is "undefined". A parameter to a function is not given, it is "undefined". If a property is not present on an object, it is "undefined".

What do you mean by REST API?

The name REST API comes from Representational State Transfer, and it follows a client-server model used to collect and deliver data over the Internet. A REST API uses a standard set of HTTP methods to retrieve and return data, such as GET (to return data), POST (to retrieve new data), PUT (to update data), and DELETE (to delete data for).REST API is a software interface design principle used to collect and provide data. These principles are used as a standard protocol for collecting and delivering data on the Internet. These standards most often use the HTTP protocol, and services provided by a REST API typically provide data in JSON format.
